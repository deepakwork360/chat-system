# S1mple Chat 💬

A real-time, production-ready workspace chat application.

## Features
- ⚡ Real-time messaging via Socket.io WebSockets
- ✅ Read receipts with blue double-ticks
- 🎙️ Voice notes with custom audio player
- 📎 File & image attachments  
- 👥 Group chats with member management
- 📱 Fully mobile-responsive
- 🔐 JWT-based session authentication

## Stack
- **Frontend**: Next.js 16, TypeScript, TailwindCSS, Framer Motion
- **Backend**: Node.js, Express, Socket.io
- **Database**: PostgreSQL
